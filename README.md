# CineMate

A user-friendly movie discovery platform that fetches data from the TMDB API.  
Quickly search for movies, apply filters, and see streaming availability.

## Features
- Search for movies by title.
- Filter by genres and sort by popularity, rating, release date, or revenue.
- View detailed movie info in a modal, complete with trailers and streaming providers.

## Getting Started
1. Clone or download this repository.
2. Obtain a TMDB API key from [The Movie Database](https://www.themoviedb.org/).
3. Insert your TMDB API key into the script.js file (replace "YOUR_API_KEY_HERE" if needed).

## Usage
- Open index.html in your browser.
- Use the search bar and custom selects to find and filter movies.
- Click a movie card to open details in a modal.
